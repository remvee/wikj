# wikj

A tiny wiki.

## Prerequisites

You will need [Leiningen][1] 1.7.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

Run:

  lein uberjar
  java -jar target/wikj-*-standalone.jar "My Site" --port 3000 --store-file /var/lib/wikj.edn
